## About The Project

Breedle is a Wordle-style web game where the player must guess the dog breed based on an image.
You have 6 attempts to guess correctly.

## Features

- Random dog breed generator (API based)
- Dynamic image fetching
- Word length display
- Guess history tracking
- Win / Lose screens
- Glassmorphism modern UI design
- Responsive layout
